# Sample Loom Transcript

Use this as a template for manually pasting transcripts into the app.

## Format

Each line should have a timestamp followed by the feedback/task:

```
0:05 - The header is misaligned on the home page
0:15 - Button color should be blue instead of green  
0:23 - Add a loading spinner to the submit button
0:45 - Footer links are broken and need to be fixed
1:02 - Profile image is not uploading correctly
1:18 - Search bar needs autocomplete functionality
```

## Alternative Formats (also supported)

With colon separator:
```
0:05: The header is misaligned
```

Without separator:
```
0:05 The header is misaligned
```

## Tips

1. **Be specific**: Include screen names, URLs, UI elements
2. **One task per line**: Each line should describe one distinct issue
3. **Use timestamps**: Format as M:SS or MM:SS (e.g., 0:05, 1:23, 12:45)
4. **Include context**: Mention what page, what action, what should happen

## Example Full Transcript

```
0:08 - On the dashboard, the "New Project" button is not centered
0:22 - The sidebar navigation needs a search feature
0:35 - User profile page: avatar upload is failing with large images
0:51 - Settings page needs a dark mode toggle
1:12 - The pricing table has misaligned columns on mobile
1:30 - Contact form is missing email validation
1:45 - Team page: member cards should have hover effects
2:03 - Documentation links in footer are returning 404 errors
```

## Getting Your Transcript

### From Loom UI (Manual Method)
1. Open your Loom video
2. Click the "..." menu
3. Select "Download SRT" or "Download VTT"
4. Open the file and copy the text
5. Format it as shown above (remove extra SRT formatting)

### From Loom Share Page
1. Open the video in a browser
2. Right-click on the video
3. Some browsers allow "Copy transcript"
4. Paste and format as shown above
